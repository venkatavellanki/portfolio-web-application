# Portfolio Website – Venkat Aditya

## Project Overview
A modern, responsive portfolio website designed to showcase personal skills, projects, and contact information.  
The project demonstrates proficiency in **frontend development, web design, and aesthetic presentation** suitable for a fresher-level professional profile.

## Technologies Used
- HTML5  
- CSS3  
- Bootstrap 5  
- JavaScript  

## Key Features
- Hero section with gradient background and call-to-action buttons  
- Skills section highlighting frontend technologies with interactive hover effects  
- About page with professional background and career summary  
- Projects page featuring:
  - **Sight Sense** – AI-based automated image captioning system  
  - **FortiGuard** – AI-driven DDoS, DoS, MITM attack detection and mitigation  
  - **Banking System** – Secure data and transaction management  
  - **Airline Passenger Management** – Efficient passenger data handling  
- Contact page with a clean, professional form  
- Consistent modern design with responsive layout and visual appeal
